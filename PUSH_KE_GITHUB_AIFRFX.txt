Buka CMD pada folder proyek ini, lalu jalankan:

cd "LOKASI_FOLDER_MCU_JP_FINAL_GITHUB_BACK4APP"
git init
git branch -M main
git add .
git commit -m "Initial final MCU application"
git remote add origin https://github.com/aifrfx/mcu-jp.git
git push -u origin main

Jika remote origin sudah ada:

git remote set-url origin https://github.com/aifrfx/mcu-jp.git
git push -u origin main

Jika repository GitHub sudah berisi commit:

git pull --rebase origin main
git push origin main
